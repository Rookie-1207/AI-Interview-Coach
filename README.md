# AI Interview Coach

A full-stack AI-powered interview preparation platform.

## Features

- User Authentication (JWT)
- Resume Upload & Parsing
- AI Interview Question Generation
- MongoDB Atlas Database
- React Frontend
- Express Backend
- Gemini AI Integration
- Render Deployment

## Tech Stack

### Frontend
- React
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Multer
- PDF Parse

### AI
- Google Gemini API
